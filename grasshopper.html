Using Grasshopper

This section will provide a brief summary of the user interfaces for both the Grasshopper plugin, as well as its host Program, Rhino. As of Rhino 6, Grasshopper is included with the software by default. For Rhino versions 4 and 5, a download link can be found here. In order to use these versions, you must provide your email. Plugins available for Grasshopper itself, as well as other Rhino plugins can be found here.

The Rhino Interface

Although intimidating at first glance, the Rhino interface can easily be broken down into 4 main section. These sections,as seen in the figure bellow, are the toolbar (1), sidebar (2), the viewports(3), and panels (4). Since it is not the main focus of this tutorial, this interface will only be explained up to the point needed to use Grasshopper. For more information on rhino itself, the official tutorial seres can be found here.

The toolbar is where projects can be saved and loaded, and settings can be changed, as in most programs. However, the Rhino toolbar includes a command line interface, where the user can type in any of the different function Rhino includes. This interface also features and In addition to this, the toolbar offers several different kinds of layouts, that can be selected and customized based on what takes the user is currently preforming. For now, the default setting is all that is needed. Changing the kind of layout also changes what is available in the sidebar. The sidebar offers quick access to a suite of tools most relevant to the toolset the user has selected.

The viewport is the main way to interact with the 3D models generated by Grasshopper directly in Rhino. To focus in on an individual view, double click the tab in the top right corner. Double clicking this again will return the viewport to its split view. Left click is used to select objects in the view. By clicking and dragging, it is possible to select multiple objects by drawing a box around them. If the box is drawn from left to right, it will select everything the box is touching. If done from right to left, it will only select what is inside the box. Middle click opens up a quick select menu, similar to the radial menu in Maya. In the perspective view, right click rotates the view, scroll wheel zooms in and out, and holding the shift key pans the view. In the other three views, it is imposable to rotate the views, so right click pans the view on its own.

The panels on the right of the interface can be customized to displayed several different useful pieces of information, and can be customized by the user by right clicking the top of that window. By default, it displays information on display settings and layers, as well as a link to the official help page. Any panel can be moved around the interface by clicking and dragging its nametag, and placing it along the edge of another interface element.
